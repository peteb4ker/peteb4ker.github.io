d3.chart.ArcLabels Class

Extends d3.chart.BaseChart
Defined in: bower_components/d3.chart.arclabels/d3.chart.arclabels.js:6

Arc labels control for the d3.chart framework.

Renders a set of labels around an arc. This control is designed to be used as a mix-in component to a parent chart.

Basic usage

chart.arcLabelsChart = chart.append("g").chart("ArcLabels")
                                        .text(textFunction)
                                        .arc(arc);
