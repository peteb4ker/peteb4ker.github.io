d3.chart.Axis Class

Extends d3.chart.BaseChart
Defined in: bower_components/d3.chart.axis/d3.chart.axis.js:6

Axis component for d3.chart v0.2 and d3.chart.base v0.4

Intended to be used as a component of a larger chart. This mix-in can determine the axis.domain scale if domain([domain]) is set.

Basic usage

chart.xAxisMixin = chart.append("g").chart("Axis").axis(chart.xAxis());

Customizing the CSS class and domain access method

   var yAxisPaddingScale = 1.1;

   chart.yAxisMixin = chart.append("g").chart("Axis", {
       cssClass: "y"
   }).axis(chart.yAxis())
     .domain(function(data) {
           var yExtents = d3.extent(data, function(d) { return d[1]; });
           var yMaxAbs = d3.max(yExtents.map(Math.abs)) * yAxisPaddingScale;
           return [-yMaxAbs, yMaxAbs];
   });
